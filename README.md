# DeckDrop

A mobile card game built in Godot 4.6 — Tetris meets poker.

Tap a column to drop the current card. Fill rows or columns with five cards to score poker hands (pair through royal flush, with cascade multipliers). Endless mode with a Tetris-style difficulty ramp, persistent XP/level, and a daily-seed leaderboard.
